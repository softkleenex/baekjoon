# 백준 문제 풀이

백준 온라인 저지 문제 풀이 저장소입니다.

## 📁 디렉토리 구조

각 디렉토리는 날짜별로 구성되어 있으며, 해당 날짜에 풀이한 문제들이 포함되어 있습니다.

```
baekjoon/
├── 09_02/    # 9월 2일
├── 09_03/    # 9월 3일
├── 09_10/    # 9월 10일
├── 09_13/    # 9월 13일
├── 09_21/    # 9월 21일
├── 09_25/    # 9월 25일
├── 09_27/    # 9월 27일
├── 09_28/    # 9월 28일
├── 09_30/    # 9월 30일
├── 10_01/    # 10월 1일
├── 10_10/    # 10월 10일
├── 10_11/    # 10월 11일
├── 11_11/    # 11월 11일
└── 11_13/    # 11월 13일
```

## 💻 사용 언어

- Python

## 📌 문제 풀이 방식

각 파일명은 백준 문제 번호로 구성되어 있습니다.
- 예: `22860.py` → [백준 22860번](https://www.acmicpc.net/problem/22860)
